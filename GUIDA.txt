Ciao! Benvenuto in pfinance.

Questa guida ti aiuterà a configurare e avviare l'applicazione per la prima volta.

**Prerequisiti:**

Per eseguire questa applicazione, hai bisogno di Node.js e npm. npm è il gestore di pacchetti di Node.js e viene installato automaticamente con esso.

**Come installare Node.js e npm:**

1.  **Verifica se Node.js e npm sono già installati:**
    Apri il tuo terminale (o Prompt dei comandi su Windows) e digita i seguenti comandi:
    ```
    node -v
    npm -v
    ```
    Se vedi dei numeri di versione, significa che sono già installati e puoi passare alla sezione "Avvio dell'applicazione".

2.  **Installa Node.js (che include npm):**
    - Vai sul sito ufficiale di Node.js: https://nodejs.org/
    - Scarica la versione LTS (Long Term Support), che è la più stabile.
    - Esegui il file di installazione scaricato e segui le istruzioni a schermo.

**Avvio dell'applicazione:**

Una volta che hai Node.js e npm installati, segui questi passaggi:

1.  **Apri un terminale** nella cartella principale di questo progetto.

2.  **Installa le dipendenze del progetto:**
    Esegui questo comando per scaricare tutte le librerie necessarie:
    ```
    npm install
    ```

3.  **Avvia l'applicazione in modalità sviluppo:**
    Esegui questo comando:
    ```
    npm run dev
    ```

4.  **Apri l'applicazione nel browser:**
    Apri il tuo browser e vai all'indirizzo: http://localhost:3000

Ora dovresti vedere l'applicazione in funzione!

**Comandi utili:**

- `npm run build`: Per compilare l'applicazione per la produzione.
- `npm run start`: Per avviare il server di produzione dopo la compilazione.
- `npm run lint`: Per controllare la qualità del codice.

Spero questa guida ti sia utile. Buona fortuna!
